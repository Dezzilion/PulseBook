# Backend

NestJS мікросервіси для PulseBook.

Сервіси:
- `auth-service` — реєстрація, авторизація, профілі користувачів
- `event-service` — управління подіями та залами
- `booking-service` — бронювання та тимчасове блокування місць
- `payment-service` — оплати та генерація квитків
- `analytics-service` — звітність та статистика
